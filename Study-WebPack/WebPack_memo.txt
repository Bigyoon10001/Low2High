npm 초기화

- npm init -y : 기본값으로 바로 생성

npm 라이브러리 설치
- npm install <라이브러리>

npm을 사용하는 이유
1. package.json에 사용 라이브러리를 명시함으로서 버전관리와 의존성 관리에 용이하다.
2. 설치가 쉽다?


지역설치 vs 전역설치
- 지역설치 : 해당 프로젝트의 node_modules에 설치됨
    - 개발용:  서비스 로직에 필요하면 사용
- 전역설치(--global or -g) : 시스템 레벨 경로에 설치됨
    - dev용: 서비스 로직에 직접적인 영향이 없을 때 사용

웹팩이란?
- 최신 프론트에드 프레임워크에서 가장 많이 사용되는 모듈 번들러
    - 모듈 번들러: 웹 애플리케이션 구성하는 자원(HEML, CSS, Javascript, Images)을 각각의 모듈로 보고 이를 조합해서 병합된 하나의 결과물을 만드는 도구 

모듈 번들링
- 애플리케이션을 구성하는 몇십, 몇백개의 자원들을 하나의 파일로 병합 및 압축 해주는 동작
- 빌드 = 번들링 = 변환
